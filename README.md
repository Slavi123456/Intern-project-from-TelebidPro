# Проекти

Това са разработени проекти през периода ми във стажанската програма на компанията TelebidPro

Първият е "Chitanka" - конзолен web scrapper, който  изтегля данните от https://chitanka.info 
и ги записва в PostgreSQL база данни. Програма извежда и отчет за успешно заредените данните.

Вторият е "Ekatteproject" - Този проект зарежда и вкарва данни от официалните ЕКАТТЕ файлове, 
предоставени от Националния статистически институт (NSI EKATTE), в PostgreSQL база данни.
Освен това предоставя уеб интерфейс за търсене,създаване,триене и обновяване на информация за селища. 
С възможностите да се сортират по колони и да се изтегля статистика с текущия брой селища, общини, кметства и области.
