# Chitanka Project

    Този проект изтегля данните от https://chitanka.info и ги записва в PostgreSQL база данни.
    Програма извежда и отчет за успешно заредените данните.

## Oсновни технологии

- **Node.js** – За backend логиката.
  - **Аxios** - за HTTP заявки и изтегляне на HTML страници
  - **Cheerio** - за парсване на HTML и извличане на данни чрез селектори
  - **Adm-zip** - за работа със ZIP файлове

## Key features

    * Автоматично изтегляне, структуриране и подготовка на данни за запис в базата
